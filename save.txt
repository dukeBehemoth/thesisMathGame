Сохранение прогресса, статистики и настроек в MathQuiz (Godot 4.6)
====================================================================

Проект использует встроенный механизм FileAccess для сохранения данных
в директории user:// (на разных ОС это AppData/ или эквивалент).
Формат хранения — бинарный (store_var / get_var).

Все файлы сохраняются в user://:

1. stats.save — Статистика игрока (StatsManager.gd)
   Сохраняемые поля:
   - total_games_played (int) — всего сыграно игр
   - total_correct (int) — всего правильных ответов
   - total_wrong (int) — всего неправильных ответов
   - best_score (int) — лучший результат за одну игру
   - best_streak (int) — лучшая серия правильных ответов
   - total_days_played (int) — количество дней с игрой
   - last_play_date (String) — дата последней игры (YYYY-MM-DD)
   - section_stats (Dictionary) — статистика по каждому разделу:
       { "algebra": { "correct": 10, "wrong": 2 }, ... }
   Сохранение происходит:
   - после каждой игры (record_game)
   - после каждого ответа (record_answer)
   - при первом запуске дня (_check_daily)

2. achievements.save — Достижения (AchievementsManager.gd)
   Хранится Array[String] — список id открытых ачивок.
   Сохраняется при открытии каждой новой ачивки.

3. settings.save — Настройки (SettingsManager.gd)
   Сохраняемые поля:
   - sound_volume (float) — громкость звука (0.0–1.0)
   - music_volume (float) — громкость музыки (0.0–1.0)
   - difficulty (int) — сложность (0=EASY, 1=MEDIUM, 2=HARD)
   - questions_per_game (int) — количество вопросов за игру (1–100)
   - enabled_sections (Array[String]) — выбранные разделы
   Сохраняется при любом изменении настроек (apply_settings).

4. theme.save — Тема оформления (ThemeManager.gd)
   Сохраняемые поля:
   - version (int) — версия схемы (для миграции)
   - is_dark (bool) — тёмная тема (true) или светлая (false)
   Сохраняется при переключении темы.

Важно: вопросов в файлах save нет. Они генерируются динамически
в QuestionBank.gd через генераторы (Algebra.gd, Geometry.gd и т.д.).
QuestionDownloader.gd — заглушка, не реализован.

Сброс всех данных: в настройках есть кнопка "Сбросить данные",
которая очищает все 4 файла save и сбрасывает автолоад-менеджеры
в исходное состояние.
