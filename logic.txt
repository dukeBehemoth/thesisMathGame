=== Генерация заданий в MathQuiz ===

1. **Запуск игры**
   GameManager.start_game() берёт настройки (сложность, количество вопросов,
   выбранные разделы) и вызывает QuestionBank.get_questions().

2. **QuestionBank** (Scripts/Quiz/QuestionBank.gd)
   - Для каждого включённого раздела вызывает соответствующий генератор
     (Algebra, Geometry, Percentages и т.д.) через match section_id.
   - Каждый генератор создаёт per_section вопросов (ceil(count*2 / число_разделов)).
   - Все вопросы перемешиваются, обрезаются до запрошенного count.

3. **Генераторы** (Scripts/Quiz/Generators/*.gd)
   - Каждый имеет статический метод generate(difficulty, count),
     который в цикле вызывает _gen_problem(difficulty).
   - _gen_problem использует match difficulty (0=Easy, 1=Medium, 2=Hard):
     * Случайно выбирает числа/параметры в заданном диапазоне.
     * Вычисляет правильный ответ.
     * Передаёт всё в QuestionHelpers.make_question_custom().

4. **QuestionHelpers** (Scripts/Quiz/QuestionHelpers.gd)
   make_question_custom() формирует финальный словарь вопроса:
   - Берёт правильный ответ + 3 явных дистрактора (типичные ошибки).
   - Удаляет дубликаты (если дистрактор совпал с правильным ответом).
   - Если дистракторов меньше 3 — добивает случайными числами.
   - Перемешивает варианты, запоминает индекс правильного.

5. **Структура вопроса** (словарь Dictionary):
   {
     "question_text": "текст задачи",
     "answers": ["вар1", "вар2", "вар3", "вар4"],
     "correct_index": int,         // индекс правильного варианта
     "difficulty": int,            // 0, 1 или 2
     "section": "algebra"          // ID раздела
   }

6. **Пример (Geometry, diff=0)**
   - Случайные w=5, h=3.
   - Правильный ответ: w*h = 15.
   - Дистракторы: (w+h)*2=16 (периметр), w+h=8 (сумма), w*h+h=18 (+одна сторона).
   - После дедупликации и перемешивания: ["15", "16", "8", "18"].
   - correct_index = позиция "15".

7. **Список разделов и их генераторы:**
   - algebra     → линейные уравнения, x² = n
   - geometry    → площадь, периметр, теорема Пифагора, объём
   - percentages → проценты от числа, отношение, изменение
   - probabilities → кубик, шары, монеты, dN
   - logarithms  → степени, логарифмы, показательные уравнения
   - trigonometry → sin/cos/tg табличных углов
   - derivatives → производная, точки экстремума
   - parameters  → линейные и квадратные уравнения с параметром
   - economics   → вклады, проценты, кредиты
   - number_theory → сумма цифр, делители, НОД/НОК, 2^n

8. **Отображение**
   task_screen.gd получает вопрос, показывает текст и 4 кнопки-варианта.
   После ответа — проверка, фидбек, обновление статистики.
